# Hi, I'm Tommie 👋

[English](README.md) | [日本語](README.ja.md)

[![GitHub](https://img.shields.io/badge/GitHub-@tommie--jp-181717?logo=github)](https://github.com/tommie-jp)

リアルタイム通信・3D・低レベル I/O が好きな個人開発者。
Web フロントから FPGA まで、レイヤをまたいで作るのが趣味です。

## 🛠 主なプロジェクト

- **[tommieChat](https://github.com/tommie-jp/24-mmo-Tommie-chat)** — ブラウザで動く 3D MMO チャットゲーム (Babylon.js + Nakama)
- **[Reversi UART Protocol](https://github.com/tommie-jp/31-reversi-uart-protocol)** — 自作 CPU (FPGA / ディスクリート) を PC に UART 接続するためのプロトコル仕様
